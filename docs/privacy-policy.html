FaciBox — Privacy Policy

Facilitation Toolkit for Workshops & Meetings

Overview

FaciBox is a Chrome extension that provides facilitation tools (dice, roulette, timer, theme picker) for workshops and meetings. We are committed to protecting your privacy.

Data Collection

FaciBox does not collect, transmit, or share any personal data.

  • No personal information is collected
  • No usage analytics or tracking
  • No cookies
  • No network requests to external servers

Local Storage

FaciBox stores your settings and data (dice configuration, roulette items, timer settings, theme lists, and presets) locally in your browser using chrome.storage.local. This data:

  • Never leaves your device
  • Is not accessible to any external service
  • Can be cleared by uninstalling the extension

Permissions

FaciBox requests the following permissions:

  • storage — to save your settings and data locally
  • activeTab — to display the floating window on the current page
  • scripting — to inject the UI into the page

These permissions are used solely for the extension's core functionality.

Third-Party Services

FaciBox does not use any third-party services, APIs, or SDKs.

Changes to This Policy

If this policy is updated, the changes will be posted on this page with an updated date.

Contact

For questions about this privacy policy, please open an issue on GitHub.

FaciBox — プライバシーポリシー

ワークショップ・ミーティング向けファシリテーションツール

概要

FaciBox は、ワークショップやミーティングで使えるファシリテーションツール（ダイス・ルーレット・タイマー・テーマピッカー）を提供する Chrome 拡張機能です。ユーザーのプライバシー保護に努めます。

データの収集

FaciBox は個人データの収集・送信・共有を一切行いません。

  • 個人情報の収集なし
  • 利用状況の分析やトラッキングなし
  • Cookie の使用なし
  • 外部サーバーへのネットワーク通信なし

ローカルストレージ

FaciBox は、設定やデータ（ダイス設定、ルーレット項目、タイマー設定、テーマリスト、プリセット）をブラウザ内の chrome.storage.local に保存します。このデータは：

  • 端末の外に送信されることはありません
  • 外部サービスからアクセスされることはありません
  • 拡張機能のアンインストールで削除できます

権限（パーミッション）

FaciBox は以下の権限を使用します：

  • storage — 設定やデータをローカルに保存するため
  • activeTab — 現在のページにフローティングウィンドウを表示するため
  • scripting — ページに UI を挿入するため

これらの権限は拡張機能の基本機能にのみ使用されます。

第三者サービス

FaciBox は第三者のサービス、API、SDK を一切使用していません。

ポリシーの変更

本ポリシーが更新された場合、このページに変更内容と更新日を掲載します。

お問い合わせ

本プライバシーポリシーに関するご質問は、GitHub の Issue からお問い合わせください。
