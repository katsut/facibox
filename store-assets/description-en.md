# Chrome Web Store Listing (English)

## Name
FaciBox

## Short Description (132 chars max)
Facilitation toolkit for workshops and meetings. Dice, roulette, timer, and theme picker — right in your browser.

## Detailed Description
FaciBox is a Chrome extension that supports facilitation in workshops, meetings, and team activities.

Click the toolbar icon to open a floating window on top of your current page. No page navigation needed — start using it instantly.

■ 4 Tools

🎲 Dice
Roll dice with configurable faces (2–100) and count (1–10). Auto-displays total. Great for grouping or turn order.

🎡 Roulette
Add participants or topics, spin the wheel for fair random selection. Optional auto-remove for winners.

⏱ Timer
Set minutes and seconds for countdown. Counts up after reaching zero. Toggle end sound on/off. Always visible in the header so you can manage time while using other tools.

🃏 Theme Picker
Randomly pick workshop themes or discussion topics. Save preset theme lists for quick loading.

■ Features

• Works on any webpage — overlays on top of your content
• Drag to reposition — move it anywhere on screen
• Resizable window
• Collapsible — click header to minimize, timer stays visible
• Auto-save — your data persists between sessions
• Japanese / English support

■ Use Cases

• Workshop icebreakers
• Meeting facilitation
• Team building activities
• Scrum events (retrospectives, etc.)
• Training sessions and study groups

■ Privacy

All data is stored locally in your browser (chrome.storage.local). No data is sent to any external server.
