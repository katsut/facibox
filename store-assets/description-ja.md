# Chrome Web Store 掲載用テキスト (日本語)

## 名前
FaciBox

## 簡単な説明 (132文字以内)
ワークショップやミーティングで使えるファシリテーションツール。ダイス・ルーレット・タイマー・テーマ決めの4機能をブラウザ上でサッと使えます。

## 詳細な説明
FaciBox は、ワークショップやミーティングのファシリテーションをサポートする Chrome 拡張機能です。

ツールバーのアイコンをクリックするだけで、今開いているページの上にフローティングウィンドウが表示されます。ページ遷移なしで、すぐに使い始められます。

■ 4つのツール

🎲 ダイス
面数（2〜100）と個数（1〜10）を設定してサイコロを振れます。合計値も自動表示。グループ分けや順番決めに。

🎡 ルーレット
参加者やお題を登録して、ルーレットを回して公平にランダム選出。当選者の自動削除オプション付き。

⏱ タイマー
分・秒を設定してカウントダウン。終了後は超過時間をカウントアップ。終了音のON/OFF切替可。ヘッダーに常時表示されるので、他のツールを使いながら時間管理できます。

🃏 テーマピッカー
ワークショップのテーマやお題をランダムに決定。プリセット登録で、よく使うテーマリストを一括読み込み可能。

■ 特長

• どのWebページでも使える — ページの上に重ねて表示
• ドラッグで自由に配置 — 邪魔にならない位置に移動可能
• リサイズ対応 — ウィンドウ幅を調整可能
• 折りたたみ表示 — ヘッダークリックで最小化、タイマーは常時表示
• データ自動保存 — 次回起動時もデータが残ります
• 日本語 / 英語対応

■ こんなシーンで

• ワークショップのアイスブレイク
• 会議のファシリテーション
• チームビルディング
• スクラムイベント（レトロスペクティブなど）
• 研修・勉強会

■ プライバシー

データはすべてお使いのブラウザ内（chrome.storage.local）に保存されます。外部サーバーへの通信は一切行いません。
